fromScriptNumBuffer

  • fromScriptNumBuffer(buf: Buffer, fRequireMinimal: boolean, size: number): export=

  • Inherited from export=.fromScriptNumBuffer

    • Defined in huanniangstudio/workplace/project/satoplay/subject/scrypt_proj/BCP_WORK/sensible-sdk/src/bn.js/index.d.ts:585

    Create a BN from a "ScriptNum": This is analogous to the constructor for CScriptNum in bitcoind. Many ops in bitcoind's script interpreter use CScriptNum, which is not really a proper bignum. Instead, an error is thrown if trying to input a number bigger than 4 bytes. We copy that behavior here. A third argument, size, is provided to extend the hard limit of 4 bytes, as some usages require more than 4 bytes.

    Parameters

    • buf: Buffer

      A buffer of a number.

    • fRequireMinimal: boolean

      Whether to require minimal size encoding.

    • size: number

      The maximum size.

    Returns export=
